Getting Started
===============


Supported Platforms
-------------------

OS Versions Supported
~~~~~~~~~~~~~~~~~
- Armbian Stretch    
- Raspbian Stretch   
- OSMC Stretch   
- Ubuntu Bionic 

Boards Supported   
~~~~~~~~~~~~~~~~~
Any board running any of the above listed operating systems.   


Clone the Project
-----------------

``
sudo apt-get install git  
git clone https://github.com/shivasiddharth/GassistPi    
```
